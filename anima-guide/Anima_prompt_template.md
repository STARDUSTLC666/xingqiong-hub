# ANIMA3 提示词生成模板 v3.0

> 原版模板文件 · 配合 Anima 提示词指南使用

## 模板结构

```
[质量词] [主体描述] [外观细节] [服装] [姿势/表情] [场景] [光照] [构图] [风格] [额外tags]
```

## 快速参考

- **质量词**: masterpiece, best quality, high resolution
- **风格**: anime coloring, cel shading, hand-drawn
- **负面**: low quality, blurry, bad anatomy, bad hands, extra fingers, watermark, text, logo
